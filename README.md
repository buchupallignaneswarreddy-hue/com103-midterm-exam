# Weekly Expense Tracker

A simple Python program to track weekly expenses.

## How to Run

1. Make sure Python is installed (Python 3 recommended)

2. Download or clone this repository:

```
git clone https://github.com/your-username/expense-tracker.git
cd expense-tracker
```

3. Run the program:

```
python expense_tracker.py
```

## How to Use

1. Enter your name
2. Enter your weekly budget
3. Choose a category (1–5) or 0 to skip
4. Enter description and amount
5. Repeat for all entries

## Notes

* The program allows up to 4 expense entries
* Any expense over 25% of your budget will show a warning
* At the end, a summary of total spent and remaining budget is displayed
